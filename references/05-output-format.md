- Markdown 输出模板

## Markdown 输出模板

写入用户指定位置（飞书/钉钉/IMA）时的格式：

```markdown
# 下周待办事项

> 提取时间：{{YYYY-MM-DD HH:mm}}
> 提取范围：近7天（{{start_date}} ~ {{end_date}}）
> 来源渠道：{{渠道1}}、{{渠道2}}、...

---

## 由我完成

### [1] 紧急 [有截止日期事项]
- **内容**：具体做什么
- **来源**：{{渠道}} - {{发送者}} {{时间}}："{{原话摘要}}"
- **责任人**：自己
- **截止**：{{截止时间}}

### [2] [本周内事项]
- **内容**：具体做什么
- **来源**：{{渠道}} - {{发送者}} {{时间}}："{{原话摘要}}"
- **责任人**：自己
- **截止**：本周内

### [3] [持续推进事项]
- **内容**：具体做什么
- **来源**：{{渠道}} - {{发送者}} {{时间}}："{{原话摘要}}"
- **责任人**：自己
- **截止**：持续推进

---

## 团队完成（我布置的）

### [1] 紧急 [有截止日期事项]
- **内容**：具体做什么
- **来源**：{{渠道}} - {{发送者}} {{时间}}："{{原话摘要}}"
- **责任人**：{{姓名}}
- **跟进人**：自己
- **截止**：{{截止时间}}

### [2] [本周内事项]
- **内容**：具体做什么
- **来源**：{{渠道}} - {{发送者}} {{时间}}："{{原话摘要}}"
- **责任人**：{{姓名}}
- **跟进人**：自己
- **截止**：本周内

---

*本报告由周待办提取助手自动生成*
```

## 紧急标记

| 标记 | 含义 | 适用场景 |
|------|------|---------|
| 紧急 | 有明确截止日且即将到期 | 截止日在未来3天内 |
| 本周 | 本周内完成即可 | 截止日在未来3-7天内 |
| 持续 | 持续推进 | 无明确截止日期 |

## 字段说明

每条待办必须包含以下 4 个字段：

| 字段 | 必填 | 说明 |
|------|------|------|
| 内容 | 是 | 具体做什么，动作导向 |
| 来源 | 是 | 渠道 + 发送者 + 时间 + 原话摘要 |
| 责任人 | 是 | 谁做：自己 / 团队成员姓名 |
| 截止 | 是 | 截止时间：具体日期 / "本周内" / "持续推进" |

"团队完成"类额外包含"跟进人"字段（默认是自己）。

## 通知模板

通过钉钉私聊发送给责任人的格式：

```
【下周待办提醒】您好，{{姓名}}，您负责的以下事项需在下周完成：

{{事项标题}}
  来源：{{来源描述}}
  截止：{{截止时间}}
  详情请查看：{{输出文档链接}}

{{事项标题2}}
  来源：{{来源描述2}}
  截止：{{截止时间2}}

—— 本消息由周待办提取助手自动发送
```

**去重规则**：同一责任人有多个待办时，合并为一条消息发送。
